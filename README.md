# Taller: Limpieza y Normalización de Datos con Python

Repositorio oficial del taller práctico orientado a la limpieza, transformación y normalización de datos utilizando Python.

---

## 🎯 Objetivo del taller

Desarrollar habilidades para:

- Identificar problemas de calidad de datos
- Aplicar técnicas de limpieza de datos
- Gestionar valores nulos y duplicados
- Normalizar variables numéricas
- Generar datasets listos para análisis y modelado

---

## 🧠 Temáticas abordadas

- Exploración de datos
- Calidad de datos
- Manejo de valores nulos
- Eliminación de duplicados
- Estandarización de texto
- Conversión de tipos de datos
- Manejo de outliers
- Normalización Min-Max
- Estandarización Z-Score

---

## 🛠️ Tecnologías utilizadas

- Python 3
- Pandas
- NumPy
- Jupyter Notebook
- Visual Studio Code
- Git y GitHub
